# Customer Shopping Behavior Analysis

## 📌 Project Overview

This project focuses on analyzing customer shopping behavior using Python, MySQL, and Power BI. The objective was to clean and prepare customer transaction data, perform SQL-based business analysis, and create interactive dashboards to uncover actionable insights related to customer purchasing patterns, product performance, and revenue trends.

The project demonstrates an end-to-end data analytics workflow, covering data cleaning, database management, business analysis, and data visualization.

---

## 📊 Dataset

The dataset contains customer shopping information, including:

- Customer Demographics
- Product Information
- Purchase Amount
- Review Ratings
- Subscription Status
- Discount Usage
- Shipping Type
- Purchase Frequency
- Payment Method

The dataset consists of approximately 3,900 customer transactions.

## 🔍 Business Analysis

The following business questions were analyzed using SQL:

### Customer Analysis
- Identified high-spending customers who used discounts.
- Segmented customers into New, Returning, and Loyal categories.
- Analyzed spending behavior of subscribers versus non-subscribers.
- Evaluated repeat purchase behavior and subscription trends.

### Product Analysis
- Identified top-rated products based on customer reviews.
- Determined the most purchased products within each category.
- Analyzed products with the highest percentage of discounted purchases.

### Revenue Analysis
- Calculated revenue contribution by age group.
- Compared spending patterns across customer segments.
- Analyzed purchasing trends and average transaction values.

### Shipping Analysis
- Compared average purchase amounts between Standard and Express shipping methods.
